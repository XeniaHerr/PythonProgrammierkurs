# Library

Ein zentraler Ort für `typst` Code. Hier sind formatierungsregeln, Hilfsfunktionen und mehr.


## helpers
Hilffunktionen zum formatieren


## stringformat
Automatische erkennung von Strings und die entsprechende Formatierung. Erlaubt es uns großflächig Strings wie `python` zu erkennen und zu formatieren, ohne jedesmal eine Variable nutzen zu müssen.
