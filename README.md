# Programmiervorkursskript

Das Skript zum Programmiervorkurs 2026 der Fachschaft MathPhysInfo. Das Skript basierth hauptsächlich auf dem bisherigen, wird hier aber überarbeitet und auf Python angepasst.


---

## How to Build

Das Skript ist in `typst` geschrieben. Zum erstellen des Sript reicht

```
typst main.typ
```
aus.



## How to contribute
Ich hatte ursprünglich vor das ganze alleine durchzuziehen, habe aber gemerk das ich mich da sehr unsicher fühle. Deshalb freue ich mich über jede unterstützung

1. Meinung geben
Verschiedene Meinungen sind wichtig um nicht in einen subjective Bias zu geraten. Seit gerne dazu ermutigt einfach eure Meinung zum aktuellen Stand zu geben. Beteiligt euch bei den Diskussionen und bringt gerne eure eigene Meinung und Ansicht zu Themen ein.
2. Korrektur lesen
Meine Rechtschreibung ist grauenhaft und man sieht bei fremden texten eh mehr fehler. Schaut euch gerne an was bereits geschrieben worden ist und gebt bescheid wenn euch ein fehler auffällt. Egal ob Rechtschreib oder Formatierungsfehler, *_Bug Reports welcome*
3. Selber Texte Schreiben
Ich kann vermutlich nicht alles alleine Schreiben. Falls ihr Lust habt mir zu helfen könnt ihr gerne eine Lektion überarbeiten oder selber eine neue Schreiben. Vorallem beim Python Teil, der komplett neu entstehen muss ist jede Hilfe willkommen.
